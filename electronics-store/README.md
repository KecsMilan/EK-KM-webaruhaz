# Electronics Store

Welcome to the Electronics Store project! This web application allows users to browse and purchase electronic products online. Below are the details regarding the project setup, features, and usage.

## Project Structure

```
electronics-store
├── src
│   ├── index.html        # Main HTML document for the web page
│   ├── css
│   │   └── styles.css    # Styles for the web page
│   └── js
│       └── script.js     # JavaScript functionalities for ordering and cart management
├── data
│   └── products.json     # Product data in JSON format
├── package.json          # npm configuration file
└── README.md             # Project documentation
```

## Features

- **Product Browsing**: Users can view a list of electronic products with details such as name, price, and description.
- **Cart Management**: Users can add products to their cart, view the cart, and remove items if needed.
- **Simulated Payments**: Users can simulate credit card payments for their orders.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd electronics-store
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the application.

## Usage Guidelines

- Browse through the available products and click on "Add to Cart" to include items in your shopping cart.
- View your cart to see selected items and proceed to simulate a payment.
- Follow the prompts to complete the simulated payment process.
